# Tippspiel 2

The successor of [Zemke/tippspiel](https://github.com/Zemke/tippspiel) developed by [@Ribosom](https://github.com/Ribosom) and [@Zemke](https://github.com/Zemke).

## Technologies

* Spring Boot
* Kotlin
* Ember
* ES6
* PostgreSQL
* Bulma

## Features

* Standings
* Place bets
* See bets by user
* See bets by game
* football-data.org API
